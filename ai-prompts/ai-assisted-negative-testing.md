# AI-Assisted Negative API Testing

## Prompt Used
"Generate negative and edge test cases for a POST API that creates a resource using JSON payload."

## AI Suggestions Reviewed
- Missing headers
- Empty request body
- Incorrect HTTP method
- Invalid data types
- Unsupported content types

## QA Review Notes
- Selected scenarios relevant to API behaviour
- Excluded unrealistic cases
- Converted selected cases into executable Postman tests

AI was used to expand test coverage, while final decisions were made manually.
